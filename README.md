# DevSphere Platform

A backend platform built using Node.js and Express.

## Features

- REST API server
- JSON responses
- CORS enabled
- Ready for database integration
- GitHub deployment

## Tech Stack

- Node.js
- Express.js
- GitHub

## How to Run

1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open browser:
   http://localhost:5000
